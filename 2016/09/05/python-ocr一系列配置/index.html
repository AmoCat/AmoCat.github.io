python ocr一系列配置

Veröffentlicht am 2016-09-05

###安装tesseract-ocr进行图像识别，系统：OS X EI Capitan 10.11.3
安这个包废了很大功夫，一系列依赖库以及没有相关系统的中文教程，所以一点点看着英文介绍饶了很多路。这里系统整理下安装流程。

####编译工具和依赖库：

brew install autoconf
brew install autogen
brew install m4
brew install automake
brew install libtool
brew install icu4c
brew install pango
brew install cairo
brew install PIL
brew install leptonica

以及MacPorts : 

$ curl -O https://distfiles.macports.org/MacPorts/MacPorts-2.3.4.tar.bz2   
$ tar xf MacPorts-2.3.4.tar.bz2  

$ cd MacPorts-2.3.4/        
$ ./configure  
$ make  
$ sudo make install

安完MacPorts需要配置环境变量 具体参考这里

export PATH=/opt/local/bin:/opt/local/sbin:$PATH

###tesseract-ocr安装
github地址：tesseract-ocr
编译说明在https://github.com/tesseract-ocr/tesseract/wiki/Compiling

上边的依赖包如果都安好基本没什么问题，进行如下指令：

git clone git@github.com:tesseract-ocr/tesseract.git  
cd tesseract   
./autogen.sh
./configure --with-extra-libraries=/opt/local/lib  
make   
sudo make install

或

brew install tesseract

如果要安装training tool ,则用下面的方式：

sudo port install cairo pango  
sudo port install leptonica
sudo port install icu

git clone https://github.com/tesseract-ocr/tesseract/
cd tesseract
./autogen.sh
./configure \
--with-extra-libraries=/opt/local/lib \
--with-extra-includes=/opt/local/include \
LDFLAGS=-L/opt/local/lib \
CPPFLAGS=-I/opt/local/include
make
sudo make install

make training
sudo make training-install

安装的时候，可能出现的比较奇怪的一个错误是“syntax error near unexpected token `PANGO,’”，先检查一下g++,leptonica这些有没有安好，版本新不新，如果都没啥问题，删掉原来的configure文件，然后再重新编译，一般就好了。

安装完成后在cmd中输入 

tesseract -v

可得到如下信息：

tesseract 3.05.00dev
 leptonica-1.73
  libgif 4.2.3 : libjpeg 9a : libpng 1.6.24 : libtiff 4.0.6 : zlib 1.2.8 : libwebp 0.5.1 : libopenjp2 2.1.0

安装英文和中文语言包：

brew install tesseract-eng
brew install tesseract-chi-sim

brew不好使可以从https://github.com/tesseract-ocr/tessdata直接下载相应的包并剪切到/usr/local/share/tessdata/ (tessdata所在目录，可能跟这个不太一样)，中文包是chi-sim.traineddata,英文包是eng.traineddata。

更多语言可参考这里

通过

tesseract --list-langs

可查看已安装的语言包，若出错，一般是TESSDATA_PREFIX环境变量的问题，在/etc/profile中加入环境变量：
export TESSDATA_PREFIX=/usr/local/opt/tesseract/ （tesseract所在目录，可能有所不同）
